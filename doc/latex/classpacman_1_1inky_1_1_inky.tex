Get the target for the \mbox{\hyperlink{classpacman_1_1inky_1_1_inky}{Inky}} ghost in chase mode. 

\mbox{\hyperlink{classpacman_1_1inky_1_1_inky}{Inky}} uses a complicated targeting method. It uses a combination of where the avatar is heading and the current location of the Blinky ghost. It draws a vector between the Blinky ghost\textquotesingle{}s location and a point two tiles in front of the avatar. It then extends that vector\textquotesingle{}s length by two to find \mbox{\hyperlink{classpacman_1_1inky_1_1_inky}{Inky}}\textquotesingle{}s target.
