\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classpacman_1_1arena_1_1_arena}{Arena}} class is responsible for loading mazes from J\+S\+ON files. 

It also spawns the dots and power pills contained in the maze. This class extracts information from the J\+S\+ON maze file and makes it available to other objects.
