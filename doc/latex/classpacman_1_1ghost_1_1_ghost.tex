\doxysubsection{Detailed Description}
Base class for all \mbox{\hyperlink{classpacman_1_1ghost_1_1_ghost}{Ghost}} types. 

Handles duties common to all ghosts, including moving around the maze. This class expects the \textquotesingle{}target\textquotesingle{} method to be overridden by subclasses to find where the ghost should head during \textquotesingle{}chase\textquotesingle{} mode.

Set the movement mode for this ghost. 

In the game ghosts will normally alternate between \textquotesingle{}scatter\textquotesingle{} and \textquotesingle{}chase\textquotesingle{} mode. In scatter mode the ghosts will head towards to their own corner of the maze. In chase mode they will chase the avatar in their own particular way. When the avatar consumes a power pill the ghosts enter \textquotesingle{}frighten\textquotesingle{} mode and move randomly.
