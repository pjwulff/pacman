\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classpacman__game_1_1model_1_1arena_1_1_arena}{Arena}} represents the maze in the game, including the location of all the vertices, the edges between them, and the locations of objects (such as, dots, powers, and spawn locations). 

This is an abstract base class to be inherited by actual maze shapes (such as square or hexagonal).
