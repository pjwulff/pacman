Detailed Description

The overall controller for the game.

This class implements much of the game logic, but also delegates tasks to sub-controllers which this class creates.

  direction 
)

Set the desired direction for the Avatar.

This public function is provided so that the ‘View’ can specify in which direction the user would like the Avatar to move.

Parameters
directionThe desired direction, as an angle between [-pi, pi). The GameController knows nothing of input/output, and thus does not know how the user is controlling the Avatar (keyboard, mouse, joystick, etc). An angle is used in order to remain agnostic.
