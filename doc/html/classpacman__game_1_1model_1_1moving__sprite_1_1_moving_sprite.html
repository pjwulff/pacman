def pacman_game.model.moving_sprite.MovingSprite.trans_pos (   self)

Moving sprites in this game travel between vertices in the graph.

Their current position between the two is a value between 0 and 1 and is called the sprite's trans-position. This function returns it.

Returns
The sprites' trans-position.
