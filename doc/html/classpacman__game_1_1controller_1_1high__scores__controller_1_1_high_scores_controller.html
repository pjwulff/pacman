Get a list of high scores, filtered according to some parameters.

Parameters
fThe parameters by which to filter the results. Should be a dictionary mapping different game characteristics (e.g., "difficulty-easy") to boolean values, indicating if the results should be included or not.
Returns
An ordered, filtered list of high scores. Each list element is a tuple, the first element of which is the initials, the second of which is the score itself.
