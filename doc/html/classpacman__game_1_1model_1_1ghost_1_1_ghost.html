Perform an A*-search to find the shortest path between some source node and target coordinate.

This function takes into account the fact that ghosts cannot reverse direction.

Parameters
sourceThe source node from which to start the search.
targetThe target coordinates.
prevThe previously visited node. This node will be excluded from the search as ghosts cannot reverse direction.
Returns
A list of nodes representing the shortest path between the source and the target.
